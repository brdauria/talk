# AIMD Markov-modulated exit problems - Reveal.js talk

Open `index.html` in a browser. The deck uses Reveal.js 5 from jsDelivr and the Reveal MathJax plugin, following the structure of the provided example project.

Controls:
- Arrow keys or space: move through slides.
- `S`: speaker notes, when served from a local web server.
- Add `?print-pdf` to the URL to print/export as PDF from the browser.

The main talk runs through slide 21. Slides 22-23 are backup/reference slides.
